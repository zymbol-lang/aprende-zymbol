# Aprende Zymbol — Nivel Intermedio

En este nivel vas más allá de los bloques básicos: tipos de datos compuestos, funciones como valores, transformación de colecciones, manejo de errores y organización del código en módulos.

Se recomienda terminar el [nivel básico](../basico/README.md) antes de continuar.

---

## Lecciones

| # | Lección | Qué aprenderás |
|---|---------|----------------|
| 01 | Tuplas | Registros inmutables con campos nombrados, `$~` |
| 02 | Desestructuración | Desempacar listas y tuplas en una sola línea |
| 03 | Lambdas y Cierres | Funciones sin nombre, captura de entorno con `->` |
| 04 | Transformación de listas | Mapear `$>`, filtrar `$\|`, reducir `$<` |
| 05 | Tuberías | Encadenar transformaciones con `\|>` |
| 06 | Manejo de errores | Atrapar y propagar con `!?`, `:!`, `:>`, `$!!` |
| 07 | Inspección de tipos | Metadatos en tiempo de ejecución con `#?` y `#\|..\|` |
| 08 | Módulos | Organizar código en archivos con `# name`, `#>`, `<#` |
